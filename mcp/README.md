# @yama662607/obsidian-companion-mcp

MCP server package for Obsidian Companion.

## Installation

```bash
npm install @yama662607/obsidian-companion-mcp
```

## Usage

```bash
npx obsidian-companion
```

## Notes

- Bridges MCP stdio calls to the local Obsidian plugin runtime.
- Intended to be used with the companion plugin package from this monorepo.
